
# 🎮 Willkommen bei PixelPulps


### 🤝 Mitmachen & Beiträge

Wir freuen uns über alle Arten von Beiträgen – neue Ideen, Bugfixes, Features oder Verbesserungen.

### 💡 Ideen teilen

Erstelle eine Diskussion oder ein Issue, um neue Ideen oder Verbesserungsvorschläge vorzuschlagen.

### 🐛 Bugs melden

Hast du einen Fehler gefunden? Erstelle ein Issue und informiere uns.

### 🔧 Beiträge einreichen

Forke ein Repository, mache deine Änderungen und öffne einen Pull Request.
